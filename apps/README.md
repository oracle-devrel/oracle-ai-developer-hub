# Apps

Applications and reference implementations demonstrating how to build AI-powered solutions with Oracle technologies. These complete, working examples showcase end-to-end implementations of AI applications, agents, and systems that leverage Oracle AI Database and OCI services. Each application includes source code, deployment configurations, and documentation to help developers understand architectural patterns, integration approaches, and best practices for building production-grade AI solutions.

## App vs workshop

|              | Apps                                                            | Workshops                                                   |
| ------------ | --------------------------------------------------------------- | ----------------------------------------------------------- |
| **Goal**     | A working reference implementation you can run, fork, or deploy | A guided learning path that builds up a primitive at a time |
| **Shape**    | Source code + deployment configuration                          | Student notebook + complete notebook + per-part docs        |
| **Audience** | Developers shipping AI features on Oracle                       | Developers learning the patterns                            |
| **Lives in** | [`apps/`](.)                                                    | [`workshops/`](../workshops)                                |

Some workshops also ship a chat-app variant (e.g. the supply-chain demand-planning agent ships both its workshop notebooks and a live app under one folder) so you can see the wired primitives running while you build them.
