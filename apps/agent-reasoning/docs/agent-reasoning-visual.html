Agent Reasoning

11 cognitive architectures for open-source LLMs — reasoning layer for Ollama

Overview

A comprehensive reasoning layer that intercepts LLM calls and applies advanced cognitive architectures from research papers (Wei et al., Yao et al., Shinn et al.). Works with any Ollama model via a drop-in model+strategy routing convention.

Interceptor Routing
Input Parsing
Splits gemma3:270m+cot into base model gemma3:270m and strategy cot
AGENT_MAP
21 aliases mapping to 11 agent classes. E.g. cot, chain_of_thought both resolve to CoTAgent
Streaming
stream() yields text chunks; stream_structured() emits typed StreamEvent objects
FastAPI Gateway
POST /api/generate with NDJSON streaming. Compatible with Ollama client libraries.
11 Reasoning Strategies

Single-Call Agents (Fast)

StandardAgent
Direct generation baseline. Single LLM call, no enhancement.
1 call standard.py
CoTAgent
Chain-of-Thought: step-by-step prompting for math, logic, analysis.
1 call cot.py
LeastToMostAgent
Decomposes into easy-to-hard sub-questions, solves progressively.
n+1 calls least_to_most.py

Multi-Path Exploration (Slow)

ToTAgent
Tree of Thoughts with BFS pruning. Width=2, depth=3 branching.
O(w×d) calls tot.py
ConsistencyAgent
Self-Consistency: k-sample voting (default k=5) for objective answers.
5 calls consistency.py
DecomposedAgent
Problem decomposition into sub-problems, parallel solving, synthesis.
n+2 calls decomposed.py

Iterative Refinement (Slow)

SelfReflectionAgent
Draft, Critique, Refine loop. Up to 5 iterations of self-improvement.
2-10 calls self_reflection.py
RefinementLoopAgent
Generator, Critic (score 0-1), Refiner loop. Threshold: 0.9.
up to 20 calls refinement_loop.py
ComplexRefinementLoopAgent
5-stage pipeline: outline, research, draft, refine, polish.
up to 30 calls complex_refinement.py

Tool-Using & Code Agents

ReActAgent
Reason+Act with tools: web_search, calculate, search. Up to 5 steps.
up to 5 calls react.py
RecursiveAgent
Python REPL + sub_llm() calls for data processing and long-context tasks.
up to 8 calls recursive.py

Benchmark Results

Accuracy evaluated across 4 standard NLP datasets using gemma3:latest (4.3B Q4_K_M). Chain-of-Thought achieves the highest average accuracy at 87%.

Strategy GSM8K MMLU ARC-C HellaSwag Average Rating

Self-Reflection 66.7% 90.0% 88.0% 90.0% 83.7% High

Self-Consistency 76.7% 96.7% 92.0% — 66.3% Medium

Evaluated on gemma3:latest (4.3B Q4_K_M) — OCI A10 GPU ($1.28/hr)
Benchmark datasets: GSM8K (30 math problems), MMLU (30 knowledge questions across 57 subjects), ARC-Challenge (25 science reasoning), HellaSwag (20 commonsense completion). All benchmarks run via the AccuracyBenchmark class with automated answer extraction.

Request with strategy:
curl http://localhost:8080/api/generate -d '{"model":"gemma3+cot","prompt":"Why is the sky blue?"}'

Response format (NDJSON):
{"model":"gemma3+cot","response":"chunk","done":false}
{"model":"gemma3+cot","response":"","done":true}

bool — True when updating an existing item (e.g. score change)
