# Partners · CopilotKit

This directory hosts CopilotKit partner content for the Oracle AI Developer Hub —
cookbooks that build agentic, memory-augmented chat experiences on **Oracle AI
Database** with [CopilotKit](https://www.copilotkit.ai/) as the frontend
interaction layer, over the open [AG-UI](https://docs.ag-ui.com/) protocol.

| Cookbook | Description | Stack | Link |
| --- | --- | --- | --- |
| CopilotKit × Agent Spec × Memory | An **agentic travel app**: a portable Oracle **Agent Spec** agent on LangGraph, served over AG-UI to a CopilotKit chat, with **Oracle Agent Memory** (durable recall + supersession) and human-in-the-loop booking. | CopilotKit, Agent Spec, LangGraph, AG-UI, `oracleagentmemory` | [![View Cookbook](https://img.shields.io/badge/View%20Cookbook-blue?style=flat-square)](https://github.com/CopilotKit/oracle-cookbook) |

Each cookbook is a standalone, runnable project with its own README. Written guides
are also published on the CopilotKit docs at
[docs.copilotkit.ai/cookbook](https://docs.copilotkit.ai/cookbook).
